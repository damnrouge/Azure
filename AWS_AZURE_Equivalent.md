# AWS vs Azure Service Comparison

```Mermaid

## **Compute Services**
| **AWS Service**                      | **Azure Service**                     | **Description**                                           |
|--------------------------------------|---------------------------------------|-----------------------------------------------------------|
| **EC2 (Elastic Compute Cloud)**      | **Azure Virtual Machines (VMs)**      | Both provide scalable compute instances for running applications. |
| **Lambda**                           | **Azure Functions**                   | Serverless compute services for running code without managing servers. |
| **Elastic Beanstalk**                | **Azure App Service**                 | Managed service for deploying web apps, APIs, and services. |
| **Amazon Lightsail**                 | **Azure Virtual Machine Scale Sets**  | Easy-to-use compute service for small businesses and developers. |

## **Storage Services**
| **AWS Service**                      | **Azure Service**                     | **Description**                                           |
|--------------------------------------|---------------------------------------|-----------------------------------------------------------|
| **S3 (Simple Storage Service)**      | **Azure Blob Storage**                | Object storage for unstructured data (files, backups, etc.). |
| **EBS (Elastic Block Store)**        | **Azure Disk Storage**                | Persistent block storage for virtual machines.             |
| **Glacier**                          | **Azure Blob Storage - Archive Tier** | Low-cost storage for archival data.                       |
| **FSx (Managed File Storage)**       | **Azure NetApp Files**                | Managed file storage service for shared file access.      |

## **Networking Services**
| **AWS Service**                      | **Azure Service**                     | **Description**                                           |
|--------------------------------------|---------------------------------------|-----------------------------------------------------------|
| **VPC (Virtual Private Cloud)**      | **Azure Virtual Network (VNet)**      | Virtual networks for isolating resources and controlling traffic. |
| **Elastic Load Balancer (ELB)**     | **Azure Load Balancer**               | Distributes incoming traffic across multiple servers.      |
| **Amazon Route 53**                  | **Azure DNS**                         | Managed Domain Name System (DNS) service.                 |
| **Direct Connect**                   | **Azure ExpressRoute**                | Dedicated private connection between your data center and cloud. |
| **CloudFront**                       | **Azure CDN**                         | Content delivery network for faster delivery of content.   |

## **Database Services**
| **AWS Service**                      | **Azure Service**
```
